Getting started - how to get this project to compile

= = = 

Install the Android SDK. See the NOTICE file for download location.
Import the project as an Android project.

Download the ActionBar Sherlock kit. See the NOTICE file for download location.
See this http://www.slideshare.net/androidstream/action-bar-sherlock-tutorial - take not of the third slide
You need to call the library ABS, as the DomDisc project references the library as ABS.
If you have pulled the DomDisc code from GIThub using a GIT client, you should make sure that the ABS
library is also stored in Eclipse as a GIT project. You do not need to submit ABS to GitHub, you
just need to make sure that it is stored next to DomDisc.
If you get problems with the reference to ABS, you need to look in the Properties for the DomDisc
project (Right click on the Project name->Properties->Android). Here, at the bottom you will see 
the reference to the ABS library.
